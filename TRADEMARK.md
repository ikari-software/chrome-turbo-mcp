- **Use the code** under the Apache 2.0 terms — fork, modify, ship, embed,

- **Link** to the project's GitHub or releases page using the name as the
  link text.

- **Distribute a fork or modified build under the name "TurboWeb MCP"** or
  any name that implies an endorsement or affiliation. If you ship a
  modified version, rename it.
- **Reuse the cursor mark, the icon set, or the hero composition** in
  another product's branding, marketing, or store listing.
- **Use the name to promote unrelated products or services.**

## Why

The on-page robot cursor and the "TurboWeb MCP" badge are designed to be
the visible safety signal that the user is running the real extension.
Reserving the mark and the visual identity is what gives that signal
meaning — a malicious fork can copy the code, but it cannot call itself
TurboWeb MCP or wear its badge without infringing.

If you want to do something not covered above, open an issue and ask.
